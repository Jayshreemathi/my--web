Install CLI

Requirements

  • Windows 10 Mac, Linux
  • Node.js v12 or higher

Install:

Mac [Homebrew]

$ brew install Cracko-cli

NPM

$ npm install Cracko-cli

Yarn

$ yarn install Cracko-cli
